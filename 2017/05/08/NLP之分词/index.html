介绍

中文分词将一个汉字序列切分成一个一个单独的词。中文比拉丁语系语言要复杂，比如英文，英文以空格作为天然的分隔符，而中文由于继承自古代汉语的传统，词语之间没有分隔。但在自然语言处理中，词是最小的能够独立运用的有意义的语言成分，所以分词质量的好坏直接影响之后的自然语言处理任务。
中文分词存在挑战：

  • 歧义词
  • 未登录词

不过在《中文分词十年回顾》中指出新词带来的分词问题是歧义的5倍以上，所以说新词发现是分词面临的最大挑战。

分词技术

常见的研究方法可以分为四类：

  • 基于字典的字符串匹配方法
  • 基于语言规则的方法
  • 基于传统概率统计机器学习模型的方法
  • 基于深度神经网络的方法

中文分词技术发展大概过程：

第一阶段，从80年代到90年代中，以基于词典和人工规则的方法为主。
基于规则：

  • 短语结构文法

基于词典：

  • 正向最大匹配
  • 逆向最大匹配
  • 最少词切分法
  • 双向最大匹配法

第二阶段，从90年代中到03年，分词算法开始引入基于语料库的统计学习方法。
最典型的方法：

  • 就是基于词典全切分加上最大概率路径。

第三阶段，从03年至今，中文分词由基于词的方法开始向基于字的方法转变。当前的方法都是首先根据语料训练分词模型，然后对每一个字进行标注，最后根据标注结果来进行分词。
最典型的方法：

  • HMM: 隐马尔科夫模型(Hidden Markov Model, HMM)
  • MEMM: 最大熵马尔科夫模型(Maximum Entropy Markov Model, MEMM)
  • CRF: 条件随机场（Conditional Random Field，CRF）效果要好

CRF比HMM有更弱的上下文无关性假设，效果要好一些。
基于传统的机器学习模型的性能受限于特征的选择和提取，训练是基于人为设定的特征。

第四阶段，13年首次将神经网络模型应用到中文分词任务中，15年使用LSTM解决中文分词中的问题取得了不错的分词效果。
最典型的方法：

  • BI-LSTM-CRF
