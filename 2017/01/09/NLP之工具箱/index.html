平台

  • paddlepaddle 百度开源的DL框架
  • Scikit-learn 入门级机器学习库，浅层学习算法，包含大量的机器学习算法和特征提取实现
  • TensorFlow 用于语音识别、图像识别 等多项机器学习和深度学习领域
  • Theano 相对比较早的DL框架，常用于学术研究
  • Anaconda 用于科学计算的Python开发平台
  • Keras 高级别的Python神经网络框架，能在TensorFlow或者 Theano 上运行，中文文档

工具包

中文工具包

  • LTP 语言技术平台 by 哈工大 (C++) 中文自然语言处理工具，可以用于分词 词性标注 命名实体识别 依存句法分析 语义角色标注
  • THULAC 中文词法分析工具包 by 清华 (C++/Java/Python) 中文词法分析工具包，具有中文分词和词性标注功能
  • FudanNLP by 复旦 (Java) FNLP主要是为中文自然语言处理而开发的工具包，也包含为实现这些任务的机器学习算法和数据集
  • NLPIR by 中科院 (Java) 主要功能包括中文分词；词性标注；命名实体识别；用户词典功能
  • HanNLP (Java) 自然语言处理 中文分词 词性标注 命名实体识别 依存句法分析 关键词提取 新词发现 短语提取 自动摘要 文本分类 拼音简繁
  • SnowNLP (Python) Python library for processing Chinese text
  • YaYaNLP (Python) 纯python编写的中文自然语言处理包，取名于“牙牙学语”
  • DeepNLP (Python) 使用Tensorflow的深度学习library实现NLP Pipeline
  • BosonNLP by Boson (商业API服务)
  • chinese_nlp (C++ & Python) 中文自然语言处理工具以及examples
  • Chinese-Annotator (Python) 中文文本标注工具
  • Mallet(Java) 支持文档分类、序列标注、主题建模

英文或支持多语言的NLP工具包

  • CoreNLP by Stanford (Java) NLP工具包
  • NLTK (Python) 在NLP领域中，常使用的一个Python Toolkit
  • OpenNLP (Java) 基于机器学习的工具包，用于处理自然语言文本。
  • Gensim (Python) Gensim可实现topic modelling，document indexing和大型语料库相似性检索的Python库。 常用于从原始的非结构化的文本中，无监督地学习到文本隐层的主题向量表达
  • spaCy (Python) 工业级自然语言处理工具

分词

  • ansj_seg(Java) 基于n-Gram+CRF+HMM的中文分词。
  • Jieba (Python及大量其它编程语言衍生) 做最好的 Python 中文分词组件
  • ik-analyzer (Java) 基于java语言开发的轻量级的中文分词工具包
  • kcws (Python) 深度学习中文分词，BiLSTM+CRF与IDCNN+CRF
  • ID-CNN-CWS (Python) Iterated Dilated Convolutions for Chinese Word Segmentation
  • Genius 中文分词 (Python) 中文分词组件，采用 CRF(Conditional Random Field)条件随机场算法。
  • loso 中文分词 (Python) 根据词典数据库确定分段，并且该算法基于HMM
  • yaha “哑哈”中文分词 (Python) 基于查找句子的最大概率路径来进行分词。
  • ChineseWordSegmentation (Python) 无需语料库的中文分词

信息提取

  • MITIE (C++) information extraction工具
  • Duckling (Haskell) Language, engine, and tooling for expressing, testing, and evaluating composable language rules on input strings.
  • IEPY (Python) 专注于关系抽取的信息抽取的开源工具
  • Snorkel (Python) 一个专注于信息提取的培训数据创建和管理系统
  • Neural Relation Extraction implemented with LSTM in TensorFlow 用TensorFlow中的LSTM实现关系抽取。
  • A neural network model for Chinese named entity recognition 中文命名实体识别，基于TensorFlow实现的神经网络模型实现
  • Information-Extraction-Chinese IDCNN/biLSTM+CRF实现的中文实体识别,biGRU+2ATT实现的关系提取
  • Familia 支持用户以“拿来即用”的方式进行文本分类、文本聚类、个性化推荐等多种场景的调研和应用。
  • text_classification 通过deep learning探索NLP中的text classification方法。
